## Run Locally

Clone the project

```bash
  git clone https://github.com/avillanuevar/guidesmith-phone-catalog.git
```

Go to the client and server directory

```bash
  cd server
  or 
  cd client
```

Install dependencies

```bash
  npm install
```

Start both directories in separate consoles

```bash
  npm start
```
